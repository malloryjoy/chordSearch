## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

Chord pressers are an incredible way to play guitar with ease if you are just learning to play or if you have difficulty pressing down on strings (be it from arthritis, a neurological impairment, etc.). However, not all chord combinations are present in this tool. As such, it is a risk to select a song at random to learn, since it may require using a chord that is not available. 

This project aims to "narrow down" the list, providing a library consisting only of modern guitar pieces that use only the chords available with the chord presser!

## Usage

1. Search for a song to play on guitar 
2. Determine if this song is available for play using only chords on the Chord Presser
3. Get ready to rock!

## Notes

Ultimate Guitar does not offer an API due to copyright issues. In a different context, an API would expedite the build process, allowing me full access to their song library directly within this build. For now, I will make do with creating my own library. 

This project was inspired by my father, who played guitar from a very young age up until his late sixties. In 2022 he underwent a great deal of medical trauma, and while he has recovered to an extent, he suffered irriversible neurological damage and is unable to play as he once did. My library currently consists of music he would have played (or wanted to learn) when all he needed was the time, as well as personal favorites of my own. The email feature allows for other players to contribute so this library may grow!
